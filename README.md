# Hotel Booking Analysis Dashboard

## Project Overview

This project analyzes hotel booking data to identify booking trends, cancellation patterns, revenue drivers, and customer behavior using Excel, SQL, and Power BI.

Data cleaning and preprocessing were performed in Excel, where new columns such as **Total Nights**, **Total Guests**, and **Lead Time Category** were created to enhance the analysis. The cleaned data was then analyzed in SQL and visualized in an interactive Power BI dashboard to generate actionable business insights.
---

## Tools Used

## Key Business Questions

- Which market segments have the highest cancellation rates?
- How does booking lead time affect cancellations?
- Which deposit types help reduce cancellations?
- How much estimated revenue is lost due to canceled bookings?
- Which months generate the highest ADR and estimated revenue?
- Which distribution channels bring the most bookings?
- Which countries contribute the highest estimated revenue?
- How do monthly booking trends change over time?
- How do cancellation rates differ between first-time and repeat guests?
- Which countries have an ADR above the global average?

## Key Insights

- Online TA bookings have the highest cancellation rate.
- Portugal generates the highest estimated revenue.
- July and August are peak booking months.
- Long lead-time bookings cancel more frequently.
- Repeat guests are significantly less likely to cancel.

---

## Recommendations

- Increase direct bookings.
- Improve guest loyalty programs.
- Optimize pricing during peak seasons.
- Review OTA booking policies.
